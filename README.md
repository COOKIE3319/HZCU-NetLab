# NetLab — 计算机网络基础实验大作业程序

## 概述

NetLab.exe 是一个用于浙大城市学院**计算机网络基础与实验**课程大作业的网络通信测试工具。该程序自动执行多种网络协议的通信操作，供学生配合 **Wireshark** 抓包分析，从而完成对各类网络协议报文的实验分析报告。

程序原始形态为 `netlab.exe`，本仓库为其反编译后的 Python 源码。

---

## 工作流程

### 1. 启动与身份验证

1. 程序启动后提示用户确认已连接**校园有线网络**，并已开启 **Wireshark** 进行数据捕获。
2. 按任意键继续后，要求输入：
   - **8 位学号**（`xh_in`）
   - **4 位密码**（`ps_in`，通过"学在城院"成绩栏获取）
3. 程序从服务器 `http://10.66.56.8:8180/info.zip` 下载加密配置文件，解压后读取 `config.xlsx`，根据学号匹配该学生的个性化通信参数（含各协议的发送数据、端口等）。
4. 验证密码是否正确，不匹配则退出。

### 2. Telnet 提示

程序提示学生在新窗口中手动连接 Telnet 服务器（`10.66.56.8:8023`），完成 Telnet 通信操作后返回程序窗口继续。

### 3. 五项网络通信测试（随机顺序）

程序以**随机顺序**依次执行以下 5 项网络通信测试，每项测试完成后需按任意键继续下一项：

#### 3.1 Ping 测试（ICMP 协议）

- **模块**：`ping_cmd.py`
- **原理**：使用 Raw Socket 手动构造 ICMP Echo Request 报文（类型 8），发送到服务器 `10.66.56.8`，接收 Echo Reply（类型 0）。
- **流程**：
  1. 按照 ICMP 报文格式（类型、代码、校验和、标识符、序列号、数据载荷）打包二进制数据。
  2. 自行计算校验和（`chesksum` 函数，反码求和算法）。
  3. 通过 `socket.SOCK_RAW` 发送 4 次 Ping 请求。
  4. 统计往返时间（RTT）、丢包率等信息。
- **载荷数据**：从配置文件中读取的个性化 `ping_dat`（Base64 解码后的字符串）。

#### 3.2 HTTP 网页爬取

- **模块**：`http_cmd.py`
- **原理**：使用 `urllib` 向服务器发送 HTTP GET 请求，获取网页内容。
- **流程**：
  1. 根据学号计算目标端口：`port_http` 字典映射 `(学号第5-6位 + 第7-8位) % 10` 对应的端口号。
  2. 拼接 URL：`http://10.66.56.8:{port}/{fname}`，其中 `fname` 来自配置文件。
  3. 发送 HTTP GET 请求获取 HTML 页面。
  4. 解析 `<body>` 标签内容，对其进行 Base64 解码，验证是否包含该学号。

#### 3.3 TCP Socket 通信

- **模块**：`netlab.py` 中的 `tcp_fun()`
- **原理**：使用 `socket.SOCK_STREAM` 建立 TCP 连接，发送并接收数据。
- **流程**：
  1. 计算端口号：`6230 + (学号[5] + 学号[7]) % 10`。
  2. 创建 TCP Socket，连接到服务器。
  3. 发送个性化 `tcp_dat` 数据（Base64 解码后的字符串）。
  4. 接收服务器回复（最多 1024 字节），验证数据。
  5. 关闭连接。

#### 3.4 UDP Socket 通信

- **模块**：`netlab.py` 中的 `udp_fun()`
- **原理**：使用 `socket.SOCK_DGRAM` 进行 UDP 无连接通信。
- **流程**：
  1. 计算端口号：`6570 + (学号[5] + 学号[7]) % 10`。
  2. 创建 UDP Socket，使用 `sendto()` 发送个性化 `udp_dat` 数据。
  3. 使用 `recvfrom()` 接收服务器回复，验证数据。
  4. 关闭 Socket。

#### 3.5 SSH 远程登录

- **模块**：`ssh_cmd.py`（基于 `paramiko` 库）
- **原理**：通过 SSH 协议（端口 22）远程连接服务器并执行命令。
- **流程**：
  1. 构造用户名 `S{学号}`，密码 `N{4位密码}{学号后两位}`。
  2. 使用 `paramiko.Transport` 建立 SSH 连接。
  3. 执行命令 `uptime;who am i`，获取服务器运行时间和当前登录信息。
  4. 关闭连接。

### 4. 测试完成

所有测试完成后，程序提示：

1. 停止 Wireshark 数据捕获并保存 `.pcap` / `.pcapng` 文件。
2. 使用 Wireshark 分析保存的报文数据，撰写实验报告。
3. 登录 `10.66.56.8:5000` 填写分析结果并提交。

## 文件结构

| 文件                            | 说明                                                           |
| ------------------------------- | -------------------------------------------------------------- |
| `netlab.py`                   | 主程序入口，包含身份验证、配置加载、TCP/UDP 通信逻辑及测试调度 |
| `ping_cmd.py`                 | ICMP Ping 实现（Raw Socket 手动构造 ICMP 报文）                |
| `http_cmd.py`                 | HTTP 网页爬取（`urllib` 发送 GET 请求）                      |
| `ssh_cmd.py`                  | SSH 远程登录（`paramiko` 库封装）                            |
| `onekey.py`                   | "按任意键继续" 工具（跨平台键盘输入）                          |
| `大作业使用说明学生UTF-8.txt` | 学生端使用说明                                                 |
| `netlab.exe_extracted/`       | PyInstaller 解包后的依赖库和资源文件                           |

## 涉及的网络协议

| 协议             | 层级   | 测试项      | 可分析内容                                            |
| ---------------- | ------ | ----------- | ----------------------------------------------------- |
| **ICMP**   | 网络层 | Ping 测试   | ICMP 报文结构、类型/代码字段、校验和、TTL、RTT        |
| **HTTP**   | 应用层 | 网页爬取    | HTTP 请求/响应报文、状态码、Header 字段、TCP 三次握手 |
| **TCP**    | 传输层 | Socket 通信 | TCP 三次握手、数据传输、四次挥手、序列号/确认号、窗口 |
| **UDP**    | 传输层 | Socket 通信 | UDP 报文结构、源/目的端口、长度、校验和               |
| **SSH**    | 应用层 | 远程登录    | SSH 密钥交换、加密通信过程、TCP 连接                  |
| **Telnet** | 应用层 | 手动操作    | 明文传输特点、与 SSH 的对比                           |

## 端口计算规则

以学号 `20211234` 为例：

| 协议   | 端口计算方式                                                                | 示例     |
| ------ | --------------------------------------------------------------------------- | -------- |
| HTTP   | `port_http[(int("12") + int("34")) % 10]` → `port_http[6]` → `9632` | `9632` |
| TCP    | `6230 + (int("1") + int("3")) % 10` → `6230 + 4`                       | `6234` |
| UDP    | `6570 + (int("1") + int("3")) % 10` → `6570 + 4`                       | `6574` |
| SSH    | 固定端口                                                                    | `22`   |
| Telnet | 固定端口                                                                    | `8023` |

## 运行环境

- **Python 版本**：3.8+
- **依赖库**：`paramiko`、`pandas`、`wget`、`openpyxl`
- **网络要求**：必须连接校园有线网络，能够访问 `10.66.56.8`
- **系统要求**：Ping 测试使用 Raw Socket，可能需要管理员/root 权限
